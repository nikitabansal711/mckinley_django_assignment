# MckKinley and Rice assignment for backend developer

Introduction:
 problem statement given in backend developer assignment requiring me to develop apis 
 enabling a user to register himself , his card details for a bank account. Validating those details
 as well as depositing and withdrawing money with given constraints.

Tech stack : Django with python

### Prerequisites
- [python3.8](https://www.python.org/downloads/)
- editor used : pycharm

### initialize the project
- setup a virtual environment : python3 -m venv venv
- activate the venv : venv\Scripts\activate
- Install dependencies:

- pip3 install -r requirements.txt

- Migrate and run the server


-python manage.py makemigrations
-python manage.py migrate
-python manage.py runserver
